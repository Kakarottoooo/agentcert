# Standards Mapping

AgentCert is not an official certification authority. The repository provides a
technical evidence layer that can help teams prepare for independent security,
reliability, audit, insurance, and procurement reviews.

The current mappings are intentionally practical:

- What evidence can AgentCert produce today?
- Which external agent-assurance concern does that evidence support?
- What remains outside the current scope?

## Principle

AgentCert should avoid claiming compliance it cannot independently grant. The
stronger position is narrower:

> AgentCert produces independent, reproducible evidence that another party can
> inspect, verify, and map to its own standard.
