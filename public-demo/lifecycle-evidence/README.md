# AgentCert Lifecycle Evidence

This directory contains the checked-in public demo evidence that feeds AgentCert Monitor.

- `mcpbench-passing/`: a deterministic MCPBench pre-release run for an MCP/tool server.
- `../browser-agent-robustness/evidence/tripwire-public-demo/`: the Tripwire CI browser-agent robustness run with screenshots, DOM snapshots, and traces.
- `onegent-procurement/`: a local-only Onegent Runtime procurement approval walkthrough.

The Onegent walkthrough is demo-only. It uses a local mock ERP purchase order, does not send email, does not touch payment systems, does not scrape vendor portals, and does not store credentials.
